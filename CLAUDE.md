# Commute GPS Tracker

Personal tool for recording and comparing commute routes (walk/bus/bike) under different conditions.

## Stack
- Bun + Elysia (server)
- SQLite via better-sqlite3
- Leaflet.js (map)
- Browser PWA (no build step)
